# WhoAmI

HTML 기본 구조 및 CSS/JS 애니메이션 연습을 위한 개인 포트폴리오 페이지입니다.

## 🔗 페이지 보기

[▶ index.html 바로가기](https://pyrite9.github.io/WhoAmI/)

> GitHub Pages 배포 전이라면: 레포지토리 Settings → Pages → Branch를 `main`으로 설정해주세요.

## 📌 목적

- HTML/CSS 기본 구조 학습
- CSS 애니메이션 및 트랜지션 연습
- JavaScript DOM 조작 및 모션 구현
- Canvas API를 활용한 파티클 효과 학습

## 🛠 사용 기술

- HTML5 (시맨틱 태그, Canvas)
- CSS3 (Flexbox, Grid, Keyframes, Transition, Media Query)
- Vanilla JavaScript (DOM, Intersection Observer, requestAnimationFrame)
